Add Matroid

Creates U(n,r): all r-subsets of {1,…,n} are bases.

Layers — one line per chain level, comma-separated elements (incremental, not cumulative).
The last layer must contain all remaining elements so that ∪layers = {1,…,n}.
Rank bounds — one integer per layer, strictly increasing, last value = rank.
Example: layers 1,2 / 3,4, bounds 1,2 gives ne(4, r=2, X₁={1,2}, X₂={1,2,3,4}, R=(1,2)).

sh({1,…,n}, A): B = {b₁<…<bᵣ} is a basis iff bᵢ ≤ aᵢ for all i.

Click on empty space to add a vertex.
Drag from one vertex to another to add an edge.
Right-click a vertex or edge to delete it.
